# Joshua Portfolio App (Vite + TanStack Router)

Recruiter-focused personal portfolio for Joshua Dave E. Oropilla, implemented inside the monorepo using shared `@r6/ui` components.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Quality Checks

```bash
npm run test
npm run lint
npm run format
npm run check
```

## Portfolio Structure

- `src/main.tsx`: SPA entry point (`RouterProvider` mount)
- `src/router.tsx`: Router creation and defaults
- `src/routes/*`: File-based routes
- `src/routeTree.gen.ts`: Auto-generated by the TanStack Router Vite plugin
- `src/content/portfolio.ts`: Concise portfolio content model (hero/about/work/skills/contact)
- `src/components/HeroThreeScene.tsx`: Three.js interactive hero scene with graceful fallback
- `src/styles.css`: Dark-first design tokens and portfolio styling

## Routing

To add a route, create a new file in `src/routes`.

Example:

```tsx
import { createFileRoute } from '@tanstack/react-router'

export const Route = createFileRoute('/profile')({
  component: ProfilePage,
})

function ProfilePage() {
  return <div>Profile</div>
}
```

Use `Link` for SPA navigation:

```tsx
import { Link } from '@tanstack/react-router'

<Link to="/about">About</Link>
```

## Styling and UI

- Tailwind CSS with custom tokens in `src/styles.css`
- Shared UI primitives from `@r6/ui` (`Button`, `Card`, `Badge`, `Separator`, etc.)
- Theme mode toggle (`dark`, `light`, `auto`) stored in localStorage
- Hero-first interactive design with minimal, high-signal copy
